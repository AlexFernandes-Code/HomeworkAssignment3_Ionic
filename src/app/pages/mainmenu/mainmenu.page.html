HR-Track™

The HR-Track™ app is a tool that assist comapnies to keep track of their customers, products, orders and suppliers.

Swipe for to learn more

How does it work?

Users can use the menutab to navigate to the page they want to use.

For example a user navigates to the Product Page. From there the user will be allowed to Add a New Order, Search Existing Orders, Update an Existing Order and Delete an Existing Order.

Swipe for to learn more

Who does what?

HR-Track™ app makes use of user rights to allow different users to do different things.

A Sales Manager is able to CRUD Customers, Products, Orders and Suppliers where a Sales Employee is only able to record orders.
